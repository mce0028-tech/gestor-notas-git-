# Gestor de Notas con Git

## Descripción
Este es un proyecto desarrollado para la asignatura de Entornos de Desarrollo. Consiste en una aplicación de consola escrita en Python que permite a un usuario gestionar sus notas personales de forma sencilla.

## Funcionalidades
* **Añadir notas:** Permite escribir y guardar nuevos recordatorios.
* **Listar notas:** Muestra por pantalla todas las notas guardadas previamente.
* **Validación:** El sistema comprueba que no se guarden notas sin texto.
* **Interfaz limpia:** Incluye funciones para limpiar la pantalla y menús visuales.

## Cómo ejecutar el programa
1.  Asegúrate de tener instalado **Python 3**.
2.  Descarga o clona este repositorio.
3.  Abre una terminal en la carpeta del proyecto.
4.  Ejecuta el comando: `python main.py`
